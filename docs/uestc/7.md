
# 微电子与固体电子学院  

> 本文于2013年录入/高2012级27班王宇  

综述：

电子科技大学微电子与固体电子学院的前身是电子科技大学建校初期的无线电零件系（称为第三系），后更名为无线电材料与器件系，1989年2月，原三系划分为两个单位：半导体材料与器件专业建成“微电子科学与工程系”（仍称三系）；电子材料与器件、磁性物理与器件、应用化学3个专业建成“材料工程与科学系”（十五系），1992年7月在原十五系基础上成立信息材料工程学院，下设电子材料工程系（十五系）、磁性工程系（十七系）、应用化学工程系（十八系）。2001年11月由为电子科学与工程系与信息材料工程学院合并成立微电子与固体电子学院，下设为“微电子科学与工程系”、“电子科学与技术系”、“应用化学系”。

学术氛围：

电子科大被称为电子神技高中或者清水寺职业高等技术专科学校不是没有道理的，一周五天课表都基本排满，剩下的两天总要有点事情占你一天半天的，高强度的学习的同时也有写不完的作业，所以这里如果有学术氛围的话，赶作业的氛围比较浓厚。微固学院学术氛围在整个学校相比较下算是比较浓厚的，在大一大二许多基础课程中能独享学校最优秀的老师资源，并且能够申请创新基金跟老师做项目。并且我们学院早自习制度非常严格，有效的督促我们杜绝大学散漫的生活，也让我们自己的四、六级成绩得到保障。微固学院每周四还有“微固新视

野”讲座活动，讲座主题大多和我们专业现状或未来前景之类的挂钩，也会邀请我们这个行业的佼佼者作为演讲人。在2013年中国各所高校专业排名中，电子科技大学的电子科学技术排名第一。

活动氛围：

学院自己的活动一般是由院学生会主办的，虽然说批得资金可能不是很多，但是同学们一般都比较热情比较投入，所以参加一些课外活动，还是个不错的选择。微固学院分团委除了学生会还有新闻传媒中心，星火志愿者协会，生活权益部等组织，都具有较为成熟的体制机构。不过到了大二以后，学业负担加重，很多人连海报都不会去看了，所以如果是活动主办方，抓紧每年新生刚来的那一段时间发挥发挥自己的才能。整体而言，微固学院分团委学生会仍有许多进步的空间。微固学院很少承办校团委的活动，在硬件配备上略差。

学院专业：

微电子技术、固体电子技术、集成电路设计与制造、微电子学、应用化学（备注：可能很多应届生在填报志愿时会对微电子技术和微电子学两个专业感到迷茫，所以在这里给大家声明一下。微电子学严格意义上来讲是理科，而微电子技术是工科，不过大一大二基础课程是完全相同的。微电子学的本科生可以在以后的深造中选择材料类，或者微电子技术。）
