
# 电子与信息工程学院  

> 本文于2017年录入/高2015级3班卓京港  

##综述部分

在同济大学迎来110周年校庆之际，收到《我们眼中的大学》同济大学负责人的邀请，心里简直是有一股暖流流过。能得到学弟的信任来写这篇稿子，真的很感动，但同时也让我有些焦虑，因为这不同于论文或报告只关乎我个人的成绩，我在此写下的每个字都或许将对几个灵魂产生影响。



首先不得不说，也许同济是个名气比较大的学校，但是他的综合排名一直不高，全国排名基本是十几名差不多二十左右。这也意味着在这里，除了王牌专业，你很难得到最顶尖的教育，某些学院（比如我所在的电信学院）的专业很可能比不上重大、电子科大（当然我认为这可能跟同济的工科背景过于浓重有一定关系，但这仍不能改变这个事实）。所以如果很看重这点并且希望以后从事研究工作，那么这里可能不是一个很好的选择。但是作为上海滩的老三，同济的实力还是有一定保障的。

下面吐槽一点，总结起来3句话——一流的学生，二流的老师，三流的管理。这是一名同济的体育老师对学校的评价。说白了就是这里老师不够好，管理有问题，有点配不上这么高的收数。当然这么说还是有点夸张了，但确实反应出了问题。



下面说一些大家可能比较关心的问题。

同济是综合性大学！综合性大学！！

也就是说这里不只有理科生、工科生，这里还有设计与创意学院、艺术与传媒学院、经济与管理学院、外国语学院等（好像建院女生也很多），也就意味着男女比例基本正常！



学校主要在两个校区——四平路校区（本部，城里）和嘉定校区（乡下）。具体位置大家可以翻翻地图。所有大一新生都是在四平路校区的，也就是说你至少可以在上海最繁华的地方浪一年。

在此就不得不提到选择来上海读大学的优势。也许同济的教育水平跟一些内陆的大学水平差不多，而且肯定是比不上C9大学的，但是地理位置真的很重要。很明显的一个地方就是，大型活动都会选择更发达的地方举办，比如某些演出，某些商业活动，某些展览。也就是说在这里你有更多的机会去接触更广阔的世界，我相信这对我们这个年龄段的人来说，是最重要的。不管怎么说，机会的绝对数量，在这里是更多的。



再讲一讲大家比较关心的成绩评定。在同济这个奇葩的地方，绩点分布很简单，首先是5分制，然后遵循“优-5、良-4，中-3，及格-2，不及格-0”的原则。在这里没有最终老师给你的分数，只有一个孤单的等级，按照学分对绩点进行加权平均，就是你最后的绩点。这样的话有利有弊。好处就是拿高绩点更简单了，坏处就是很难和别人拉开差距，而且某一科稍微疏忽一点，和别人直接差了1绩点。对个人来说的话，就仁者见仁，智者见智了。



然后是寝室问题,大多数寝室是四人间，也有16年才翻新的豪华宿舍--西北三--有两人间和三人间。

四平路校区寝室楼都比较老了，但是常年都会翻修，所以看起来住起来都还可以。但是不同寝室楼之间的差距可能比较大，有的是上床下桌有的是上下铺，有的有独卫有的没有，有的比较大有的特别小。反正就是一个全看人品的事情。但是最让人困扰的一点是，大部分寝室都是没有热水器的，所以洗澡要去公共澡堂。就是大家电视上看到的，没有隔间的公共澡堂。有时候人多还要排队，冬天的话，情况就比较艰苦了（不过至少澡堂里面还是很暖和）。

嘉定校区，2004年投入使用，还算比较新，但是由于缺少翻修，看起来破破烂烂的。但是不管怎么说，大家都有独卫和热水器了。而且整体条件都差不多，只有个别寝室楼没有热水器，需要去公共澡堂洗澡。

其他问题就是正常水平，没有必要专门拿出来讲了，空调、洗衣机这些都是有的。而且不会断电断网。不过还是不能使用大功率电器比如锅或者吹风。



食堂，说是很好吃的，但是我想再好吃也不能满足一个重庆人的胃，不要太期待。也不用太担心吃不惯，味道还过得去，偶尔会出现特别好吃的东西。比较难得的就是同济晚上会提供大排档，烧烤、小龙虾都有的，还是很不错，即使味道一般。



校园环境对我来说基本上是满足了，四平路校区比较小，但是绿化和建筑都挺不错的，没有给人特别现代化的感觉，更像是一个古旧的校园，这一点让人觉得很舒服。嘉定校区比较大，而且更新，有一片很大的湖和一栋看起来很高端的图书馆，不过缺少一点人情味。大家可以找学长学姐要点照片来看看，这点真的不太好讲。



同济和德国这边的交流做得挺好的，毕竟同济是德国人创办的学校。学校还有中德学院这样专门为了和德国交流而建立的学院（具体情况大家还是自己查一查）。但是和其他国家的交流的话，感觉做得比较一般，交流项目比较少。



每学期都持续19周，其中1-16是教学周，17周是考查课的评比或者考试课的答疑。这里讲一下考查课和考试课。考查课更多的是选修和一些专业课，形式不限，可能是交论文、报告，或者进行presentation,也有可能是考试（做卷子），但是考查课统一在17周结束，也就是说在17周进行最后的评比（比如交论文或者进行汇报）。考试课的话会在考试周（18、19周）进行，具体安排看教务处，但一定会是考试，考试时间两个小时。



可能有同学会关心选课的问题。首先教务网站选课分为4轮，前两轮看人品，选课人数可以超过上限，结束后会踢人，但没被踢掉，就不会再掉课了，第三轮是抢，第四轮是开学之后的调整。具体的问题到时候开学了肯定会有人讲解了，或者可以找熟悉的学长学姐咨询。至于老师的问题，我个人的观点是，了解一下老师的信息是必要的，但是没必要完全按照这个来选，特别是一些“给分好”“给分不好”的评价，太主观，不可盲从。而至于老师讲的好不好这点，可以多参考一下，选到讲得好的老师，肯定利大于弊，但是没有选到，问题也不大，所谓“师傅领进门，修行看个人”。



还想讲一下关于暑假的问题。本来一般学校考完了就基本上放假了，但是同济还有教学实践周（俗称小学期），这个时间每个专业都不一样，可能是第20周开始，也有可能是先放假，再实践。实践形式也比较丰富，很多都不在学校，还是比较有意思（计算机这种在学校写代码的就不说了）。所以每年的暑假时间不是很长，只有一个月多一些（存在暑假只有一周的专业）。



我想说的基本上就是这些了，希望能对大家提供一些帮助。

其实不管最后去了那里，最重要的是看清自己选择的目的，不要为了别人的一些看法左右自己的选择。不管你选择一所学校的目的有多么低俗或者可笑，你的选择都是值得尊重的。一定要跟随自己的想法做出选择，我相信大家更想先丰富自己的精神世界，再满足自己的物质需求，相信大家不会舍本逐末。





##专业详述部分

**专业介绍**

我是电信学院计算机科学与技术的，成绩还算是比较可以的，但是没有选择转专业。当然没转专业是因为个人本来就比较喜欢，也没有必要为了其他原因转到一个陌生而又可能不太喜欢的专业去。

计算机的话，刚进来是工科试验班（计算机类），然后大二上结束的时候会分流，2/3是计算机科学与技术，1/3是信息安全。

下面列举一下主要要修的课程，参考的是我手上的培养方案（15年入学）。

四门思政课，必修，第一学期“中国近代史纲要”和“思想道德修养和法律基础”，第四学期毛概，第五学期马克思主义基本原理。

大一，主要修军事理论，体育，大学英语，高等数学B，大学物理A，物理实验，高级语言程序设计（C++），计算机科学导论，电路电子技术，离散数学。暑期是汇编语言程序设计和军训。

（高级语言只学C++，军训是在大二开始之前）

大二，主要修两门英语选修，线性代数B，概率论与数理统计，组合数学，数据结构，数字逻辑，计算机组成原理，软件开发方法，形式语言与自动机，信息论导论，算法设计与分析，人工智能原理。暑期是数据结构、计算机组成原理、软件开发方法的课程设计。

（英语选修有很多，一年学满两门即可，大二没有体育课）

大三，主要修体育，操作系统，编译原理，数据库系统原理，计算机网络，计算机系统结构，人机交互导论，软件工程。暑期是软件工程课程设计和实习。

（大三就要开始上专业选修课了，是需要选择一个课程组，修满一定学分即可）

大四，基本就没什么课了，大部分都是准备考研或者进入实验室进行相关项目。



其实我这么列举了，应该没什么概念，基本就是大一打基础，大二学专业基础，大三进入很工程理论性和实验性都比较强的部分。不过通过课程也可以看出，专业对实验（写代码）要求比较高，动手能力需要比较强。因为计算机在国内是一门工程学科，所以必须强调实验的重要性，理论部分可能强调的不多，但是一样很重要。

从课时上来看，算是课比较少的专业，我们这届整个大二选课没问题的话，可以保证每周三天假，合理安排的话，有很多时间做自己的事情。



**学术氛围**

学术氛围真的不太好讲，不是特别了解学院整体的情况，只能说说个人感觉。

自己班上的话，可能爱学和不爱学的各占一半吧，其中比较牛的人也不在少数，所以平时学习氛围还不错，上课的时候很多汇报含金量也比较高。同时学院很多创新创业项目，感觉大家提出的想法和其中的技术含量都是值得肯定的。我想这些还是可以说明问题的。



**活动氛围**

这一点就要吐槽一下了，很多选修课老师都说同济土木强，搞得学生又土又木，虽是玩笑话，但也反应出了这不太好的活动氛围。大一其实不会有问题，一是大家刚来学校，有新鲜感，二是课业压力不大，可以充分参与各种活动或工作。但是大二来到嘉定校区，情况就不太一样了。很多社团活动开展情况都不是很理想，学院自己的活动，如果难以和同学们自己利益联系到一起，也很难举办。这可能和学术氛围有一点关系吧，爱学习的人太多了，参加活动的人自然就少了，活动氛围就可能不太好。但是学校最大型的活动，比如“十大歌手”“毕业晚会”“嘉定之星”，还是很能吸引到同学们的，因为这些活动的质量确实很高。所以整体来看，问题是存在的，但也没那么不堪。



**未来出路**

即使学的是工科，但还是认为学什么就参加什么工作，可能目光有点狭隘了。现在完全就是一个天高任鸟飞，海阔凭鱼跃的时代，未来的出路基本没有什么限制。在大学时期如果能在专业基础良好的情况下，培养好自己的其他各项能力，那么工作范围真的太广了。

对于求稳的同学来说，学了计算机，基本上就是当程序员，然后接下来稳步上升。这条路的话比较稳定，而且起步工资也挺高的，可能就是有些累和枯燥。
