# 复旦大学

复旦大学，简称复旦，旧称复旦公学、国立复旦大学。创建于1905年，是中国人 (opens new window)自主创办的第一所高等学校 (opens new window)[3] (opens new window)。位于中国 (opens new window)上海市 (opens new window)，中国著名大学，名列首批211工程 (opens new window)、首批985工程 (opens new window)名单。现已发展成一所国际著名的综合性、研究型高校，目标是建设世界一流大学[4] (opens new window)。“复旦”的字面意思是“旦复旦兮（日复一日）”，表示取之不尽，用之不竭的自力更生和勤奋[5] (opens new window)。

1905年，于右任 (opens new window)、邵力子 (opens new window)等原震旦公学 (opens new window)学生脱离震旦，拥戴马相伯 (opens new window)在吴淞 (opens new window)创办复旦公学 (opens new window)。1917年始设大学部，1952年院系调整 (opens new window)后成为文理综合性大学，2000年与上海医科大学 (opens new window)合并组建新的复旦大学。今天的复旦大学拥有邯郸 (opens new window)、枫林 (opens new window)、张江 (opens new window)、江湾 (opens new window)四大校区，中山 (opens new window)、华山 (opens new window)等11家附属医院 (opens new window)以及复旦附中 (opens new window)等若干所附属学校。

# 上海交通大学

上海交通大学（英语：Shanghai Jiao Tong University，缩写：SJTU；简称：上交、上交大 或 上海交大），是位于中国 (opens new window)上海市 (opens new window)的一所具有理工特色，涵盖理、工、医、经、管、文、法、农等9个学科门类的教育部 (opens new window)直属综合性全国重点大学[4] (opens new window)，华东五校 (opens new window)之一，中国首批七所211工程 (opens new window)和全国首批九所985工程 (opens new window)院校之一。

其前身是近代中国 (opens new window)历史最为悠久的大学之一。清末洋务派政治家 (opens new window)、实业家和福利事业家盛宣怀 (opens new window)于1896年创建的南洋公学 (opens new window)，后曾易名南洋大学、上海工业专门学校，中华民国 (opens new window)大陆时期后期称为国立交通大学 (opens new window)。1959年交通大学分立，交通大学上海部分定名为上海交通大学，1999年合并原上海农学院 (opens new window)，2005年合并原上海第二医科大学 (opens new window)。

# 上海财经大学

上海财经大学是位于中国上海市的一所大学，教育部直属211工程 (opens new window)大学。以经济 (opens new window)管理 (opens new window)学科为主，经、管、法、文、理协调发展的多科性大学。

# 同济大学

同济大学（英语：Tongji University），简称“同济”，是位于中国上海市 (opens new window)的综合性大学，是中华人民共和国教育部 (opens new window)直属的全国重点大学，行政级别为副部级 (opens new window)。是“双一流 (opens new window)”(A类)、“211工程 (opens new window)”、“985工程 (opens new window)”重点建设高校，是“建筑老八校 (opens new window)”之一，同济建筑系 (opens new window)与清华建筑系 (opens new window)并列全国榜首。是环境与可持续发展合作联盟 (opens new window)、国际设计艺术院校联盟 (opens new window)、21世纪学术联盟 (opens new window)、卓越大学联盟 (opens new window)、中俄工科大学联盟 (opens new window)成员校。

同济大学的前身是1907年创办的德文医学堂，后改名同济德文医学堂；1912年与创办不久的同济德文工学堂合并，更名为同济德文医工学堂；1923年正式定名同济大学；1927年成立国立同济大学，1949年更名同济大学。

同济大学目前共设有38个学院（系）和二级办学机构，7家附属医院，5所附属中学；有全日制本科生1.7万余人，硕士研究生1.3万余人，博士研究生近5000人。有外国留学生数千人；学校占地面积约3850亩；纸本图书400余万册。

根据2018年QS世界大学排名和2018年泰晤士亚洲大学排名，同济大学均位列全国(中国大陆)第十。

# 华东理工大学

华东理工大学，简称华理，创建于1952年，是中华人民共和国教育部直属的一所具有理工特色，覆盖理学、工学、农学、医学、法学、管理学、经济学、文学、哲学、教育学、艺术学等11个学科门类的中国的一所重点大学。

办学历史可追溯到100多年前的南洋公学和震旦学院，由交通大学 (opens new window)（上海）、震旦大学 (opens new window)（上海）、大同大学 (opens new window)（上海）、东吴大学 (opens new window)（苏州）、江南大学 (opens new window)（无锡）等校化工系在1952年全国院系调整时合并组建而成，1993年2月更为现名。现拥有徐汇、奉贤、金山科技园三大校区。[1] (opens new window)

# 上海外国语大学

上海外国语大学（英语：Shanghai International Studies University，缩写 (opens new window)为SISU），中文简称 (opens new window)上外，是一所培养涉外型、复合型外语人才的多科性、国际化、高水平特色外国语大学。始建于1949年12月，最初是在时任上海市市长 (opens new window)陈毅 (opens new window)的倡议下创立的上海俄文学校 (opens new window)，首任校长是著名翻译家姜椿芳 (opens new window)。后历经华东人民革命大学附设外文专修学校、上海俄文专（修）科学校、上海外国语学院 (opens new window)等发展时期。1994年，经中国国家教委批准，正式更为现名。

如今，上海外国语大学是中国教育部 (opens new window)直属并与上海市 (opens new window)共建、进入“211工程 (opens new window)”的全国重点大学。中国一所知名的语言类大学。
