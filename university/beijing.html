# 北京大学

北京大学[注 1] (opens new window)，简称北大[注 2] (opens new window)，创建于1898年，初名京师大学堂 (opens new window)，成立之初为中国最高学府，同时也是国家最高教育行政机关，行使教育部职能，统管全国教育。中华民国建立后，校名改为北京大学校，后又改名为国立北京大学。1916年至1927年，蔡元培 (opens new window)任北京大学校长时期，“循思想自由原则、取兼容并包之义”，推行改革，把北大办成以文、理两科为重点的综合性大学，使北京大学成为新文化运动 (opens new window)的中心、五四运动 (opens new window)的策源地。1937年抗日战争 (opens new window)爆发后，北大与清华 (opens new window)、南开 (opens new window)合并组建国立西南联合大学 (opens new window)。日本战败后，北京大学回到北平。1952年院系调整 (opens new window)后，北京大学聚集了原北大、清华、燕大 (opens new window)三校的自然科学、人文学者，奠定了北大文理两科长期领先的地位。2000年，北京大学与北京医科大学 (opens new window)合并。如今北京大学是一所兼具自然科学、人文学科、社会科学、医学以及新型工科的综合性大学。北京大学现有六个校区。北京大学校址原本在沙滩红楼 (opens new window)、公主府等地，1952年院系调整后迁入位于北京西北郊的原燕京大学 (opens new window)校园（燕园）。燕园兼收中国古典韵味和西方规划理念，主要景观有未名湖 (opens new window)、博雅塔 (opens new window)等。

# 清华大学

清华大学（英语：Tsinghua University，缩写作 THU），简称清华，旧称清华学堂、清华学校、国立清华大学，是一所位于中华人民共和国 (opens new window)北京市 (opens new window)海淀区 (opens new window)清华园 (opens new window)的公立大学 (opens new window)。始建于1911年，因北京西北郊清华园得名。初为清政府 (opens new window)利用美国 (opens new window)退还的部分庚子赔款 (opens new window)所建留美预备学校“游美学务处”及附设“肄业馆”，于1925年始设大学 (opens new window)部[7] (opens new window)。抗日战争 (opens new window)爆发后，清华与北大 (opens new window)、南开 (opens new window)南迁长沙 (opens new window)，组建国立长沙临时大学 (opens new window)。1938年再迁昆明 (opens new window)，易名国立西南联合大学 (opens new window)。1946年迁回清华园复校，拥有文、法、理、工、农等5个学院。1949年中华人民共和国成立后，国立清华大学归属中央人民政府教育部 (opens new window)，更名“清华大学”；而原国立清华大学 (opens new window)校长梅贻琦 (opens new window)于1955年在台湾新竹复校 (opens new window)，仍沿用原名。

1952年，中国高校进行院系调整 (opens new window)，清华大学文、法、理、农、航天等院系外迁，吸纳外校工科 (opens new window)，转为多科性工业大学，在土木 (opens new window)、水利 (opens new window)、计算机 (opens new window)、核能 (opens new window)等领域贡献卓越，被誉为“工程师 (opens new window)的摇篮”。1978年后，逐步恢复理科 (opens new window)、人文社科 (opens new window)、经济管理 (opens new window)类学科。1999年，原中央工艺美术学院 (opens new window)并入，成立美术学院 (opens new window)。2006年，与北京协和医学院 (opens new window)合作办学，培养临床医学 (opens new window)专业学生。2012年，原中国人民银行研究生部 (opens new window)并入，成立五道口金融学院 (opens new window)。2013年，黑石集团 (opens new window)捐助成立清华大学苏世民书院及奖学金，与著名的牛津大学 (opens new window)罗德奖学金 (opens new window)及剑桥大学 (opens new window)盖兹奖学金 (opens new window)在捐助规模及名声上皆相当[8] (opens new window)[9] (opens new window)。

截至2017年12月，清华大学拥有美术馆 (opens new window)、博物馆 (opens new window)、图书馆 (opens new window)、20个学院，及近200个科研机构、5家校办产业以及一个科技园区，分别为清华控股 (opens new window)及其旗下的紫光集团 (opens new window)、同方集团 (opens new window)、诚志集团 (opens new window)、清华科技园 (opens new window)等。学校拥有固定资产超过206亿元人民币 (opens new window)，控股资产超过3500亿元人民币[10] (opens new window)[11] (opens new window)[4] (opens new window)，是985工程 (opens new window)、211工程 (opens new window)、双一流 (opens new window)高等院校[12] (opens new window)。2016年《QS世界大学排名 (opens new window)》、《世界大学学术排名 (opens new window)》均将清华大学排在中国大学首位[13] (opens new window)[14] (opens new window)。

# 北京航空航天大学

北京航空航天大学（简称：北航，英文校名：Beihang University，缩写：BUAA），原名北京航空学院，是一所主体位于北京市 (opens new window)的公立 (opens new window)高等院校 (opens new window)，是新中国第一所航空航天高等学府。隶属于中华人民共和国工业和信息化部 (opens new window)，由中华人民共和国工业和信息化部 (opens new window)、中华人民共和国教育部 (opens new window)、中国工程院 (opens new window)和北京市人民政府 (opens new window)共建。

北航创办于1952年10月25日，地处海淀区 (opens new window)学院路 (opens new window)，是20世纪50年代著名的八大学院 (opens new window)之一。北航是中华人民共和国 (opens new window)的首批重点大学之一，名列首批原211工程 (opens new window)高校和首批世界一流大学建设高校 (opens new window)(A类)之列，是原985工程 (opens new window)高校成员，亦入选珠峰计划 (opens new window)、2011计划 (opens new window)、111计划 (opens new window)、卓越工程师教育培养计划 (opens new window)，为国际宇航联合会 (opens new window)、欧洲顶尖工业管理者高校联盟 (opens new window)、中欧精英大学联盟 (opens new window)、中国西班牙大学联盟 (opens new window)、中俄工科大学联盟 (opens new window)、中俄综合性大学联盟 (opens new window)成员。

截至2018年4月，北航拥有37个学院、研究院，覆盖工、理、管、医、法、文等10个学科门类，是以工科为主的综合性研究型高等学府；北航拥有1个国家实验室、9个国家重点实验室和一系列省部级实验室，承担了“月宫一号 (opens new window)”等重大科研项目；北航还参与、组织了万维网联盟 (opens new window)等23个协会组织。

# 北京邮电大学

北京邮电大学，简称北邮（BUPT）， 是中华人民共和国教育部 (opens new window)直属、工信部 (opens new window)共建的一所高等院校，是首批“211工程 (opens new window)”高校和优势学科创新平台项目 (opens new window)高校，是一所以信息科技为特色、工学门类为主体、工管文理相结合的多科性大学，是中国信息科技人才的最重要培养基地之一。北京邮电大学在有线通信、无线通信、电子工程、计算机技术领域享有盛誉，是中国在通信信息技术领域的最高学府。

# 北京师范大学

北京师范大学（英文 (opens new window)：Beijing Normal University），简称北师大，位于北京市 (opens new window)海淀区 (opens new window)，前身为创办于1902年的京师大学堂 (opens new window)师范馆，与北京大学 (opens new window)属同源学校，于1923年升格北京师范大学，是中国 (opens new window)历史上最早成立的师范大学 (opens new window)。北京师范大学是以基础文理学科，教育学，心理学为主要特色的教育部 (opens new window)直属全国重点大学 (opens new window)，同时也是“211工程 (opens new window)”和“985工程 (opens new window)”重点建设的高校之一。曾培养出莫言 (opens new window)、刘晓波 (opens new window)等诺贝尔奖获得者,诺贝尔奖获得者数目位居中国大陆第一位。

北京师范大学是中国著名的研究型大学 (opens new window)，其学科综合实力位居中国高校前列。2002年，北京师范大学成为首批拥有自主设置本科专业审批权的6所高校之一。学科点覆盖了除军事学以外的12个学科门类，形成了综合性学科布局。根据中国教育部学位与研究生教育发展中心发布的2017年一级学科评估（第四轮）结果，北师大教育学、心理学、中国语言文学、中国史、戏剧与影视学、地理学6个一级学科获评A＋，居中国第六位，2个一级学科获评A，7个一级学科获评A-。在英国高等教育调查公司（QS）公布的2017/18世界大学排行榜中，北师大排名第256位，在中国内地高校中排名第8位。[2] (opens new window)[3] (opens new window)。

2017年，北京师范大学进入中国“世界一流大学 (opens new window)”建设A类名单，11个学科进入中国“世界一流学科 (opens new window)”建设名单[4] (opens new window)。

# 北京理工大学

北京理工大学，简称北理工、北理，是一所以国防工业 (opens new window)为学校特色，工理文协调发展的大学，现隶属于工业和信息化部 (opens new window)。学校前身为1940年中国共产党 (opens new window)所创建的第一所理工科大学——延安自然科学院 (opens new window)，后历经华北大学 (opens new window)工学院等阶段发展成为“北京工业学院”，1988年更为现名。211工程 (opens new window)中属于首批入选的重点建设学校，在985工程 (opens new window)中定位为国内一流，世界知名，首批14所校长，党委书记由中央直接任命，享受副部级待遇的高校，也是中国首批设立研究生院的高校之一。目前拥有5个国家级重点实验室，1个国家重点实验室培育基地，6个教育部重点实验室及2个工程研究中心，11个北京市重点实验室及4个工程技术研究中心，3个国防重点学科实验室。

学校本科教育由本部中关村 (opens new window)校区（同时负责研究生教育）与良乡校区 (opens new window)（主要负责本科基础教育）两部分组成，校本部校舍建筑面积111.31万平方米，良乡校区95.01万平方米。学校在河北省 (opens new window)秦皇岛市 (opens new window)建有负责少数民族 (opens new window)汉语预科 (opens new window)教育的北京理工大学秦皇岛分校 (opens new window)。另在广东省 (opens new window)珠海市 (opens new window)建有相对独立的二级学院，北京理工大学珠海学院 (opens new window)。

# 北京林业大学

北京林业大学，简称北林，原名北京林学院，创办于1952年10月16日，是中华人民共和国全国重点大学 (opens new window)之一，亦是中华人民共和国教育部 (opens new window)直属、教育部与国家林业局 (opens new window)共建的农林类高等学校，是中国“211工程 (opens new window)”重点建设大学之一。北京林业大学设有研究生院，是教育部直属、教育部与国家林业局共建的以林学、风景园林学、林业工程为特色的全国重点大学。

北京林业大学的法定住所为北京市 (opens new window)海淀区 (opens new window)清华东路 (opens new window)35号。

# 北京化工大学

北京化工大学创办于1958年，原名北京化工学院，是中华人民共和国为“培养尖端科学技术所需求的高级化工人才”而创建的一所大学，为教育部直属的全国重点大学，国家“211 工程”和“‘985’优势学科创新平台”重点建设院校。2004年成为第一所通过教育部本科教学评估结果为优秀的高校。[1] (opens new window)

# 北京外国语大学

北京外国语大学（英语：Beijing Foreign Studies University, BFSU），简称北外，是一所中华人民共和国教育部 (opens new window)直属的语言类高等院校，创建于1941年，是国家“211工程 (opens new window)”、优势学科创新平台项目 (opens new window)重点建设高校，其前身是中国人民抗日军政大学 (opens new window)三分校俄文大队基础上创办的延安外国语学校 (opens new window)。北京外国语大学是中国最著名的语言专业类学府，学校占地面积49.2万平方米，目前已开设98种外国语课程，培养出诺贝尔奖得主 (opens new window)一人，驻外大使400余人，参赞近1000人，并因此赢得了“共和国外交官摇篮”的美誉[1] (opens new window)。

# 北京语言大学

北京语言大学成立于1962年，是中华人民共和国 (opens new window)教育部直属高等院校[1] (opens new window)，中国唯一一所以教外国人汉语为主要特点的高等院校，其前身为北京语言学院。[2] (opens new window)现在，该校不仅教授外国学生中文 (opens new window)和中国文化 (opens new window)，而且也教授中国学生外语 (opens new window)和外国文化。

校园位于北京 (opens new window)市海淀区 (opens new window)学院路15号，靠近北京大学 (opens new window)和清华大学 (opens new window)，传统上被认为是中国 (opens new window)国内对外汉语 (opens new window)教学水平最高的大学 (opens new window)，同时该校对于中国学生进行的外语 (opens new window)教育的水平也在全国名列前茅。目前，北京语言大学的外国学生人数与中国学生人数不相上下，该校不仅拥有中国国内人数最多的韩国 (opens new window)和日本 (opens new window)留学生 (opens new window)群体，还有大批来自西方国家 (opens new window)、中东 (opens new window)地区以及非洲 (opens new window)的留学生，非常国际化，被誉为“小联合国”。[3] (opens new window)以非汉语母语为对象的汉语水平考试 (opens new window)（HSK）的考试中心曾设于该校。北京语言大学出版社 (opens new window)以出版对外汉语类教材而著称。[4] (opens new window)

# 对外经贸大学

对外经济贸易大学（University of International Business and Economics），简称“对外经贸大学”、“贸大”，英文简称“UIBE”，坐落于中华人民共和国首都北京市区。学校始于1951年创建的贸易部高级商业干部学校，1954年合并组建为北京对外贸易学院，1960年列入首批64所全国重点大学之一；1984年，学校正式更名为对外经济贸易大学，因当时主管其的部门“中华人民共和国对外经济贸易部”得名。2000年6月，原中国金融学院与原对外经济贸易大学合并为现今的对外经济贸易大学。

# 中国人民大学

中国人民大学（英语：Renmin University of China, 缩写为RUC），简称人大。位于北京市 (opens new window)，是一所以人文社会科学 (opens new window)为主的综合性研究型大学，直属于中华人民共和国教育部 (opens new window)，由教育部与北京市 (opens new window)共建。

该校前身为中国共产党 (opens new window)在抗日战争 (opens new window)时期创办于延安 (opens new window)的陕北公学 (opens new window)，以及后来的华北联合大学 (opens new window)和华北大学 (opens new window)。1950年10月3日，以华北大学为基础合并组建的中国人民大学正式开学。1954年，在教育部《关于重点高等学校和专家工作范围的决议》中，与北京大学 (opens new window)、北京医科大学 (opens new window)、清华大学 (opens new window)、中国农业大学 (opens new window)、哈尔滨工业大学 (opens new window)等5所院校一同被指定为最早一批全国性重点大学。

中国人民大学是“985工程 (opens new window)”和“211工程 (opens new window)”重点建设的大学，属于中国著名代表高校，是当代中国人文社会科学高等教育和研究的重要基地，被誉为“中国人文社会科学的一面旗帜”。

中国人民大学也是双一流 (opens new window)重点建设的大学，共有14个学科入选双一流 (opens new window)重点建设学科，在全国高校中名列第七，入选学科中，有十三个人文社会学科。

人大与现位于台北市的国立政治大学 (opens new window)的性质及历史背景相仿，而且皆是以发展人文学科及社会科学领域为主的大学，并各自曾为中国共产党 (opens new window)及中国国民党 (opens new window)党校，两校也培养出不少政治家 (opens new window)及首席执行官 (opens new window)，现两校合作关系密切。[2] (opens new window)[3] (opens new window)[4] (opens new window)

2017年，人大通州 (opens new window)东校区获得批准，将成为人大最大的校区，未来人大将发展理工 (opens new window)及医学 (opens new window)领域。[5] (opens new window)

# 中央财经大学

中央财经大学，简称中央财大或中财，是所位于中国北京市的高校，为中华人民共和国 (opens new window)教育部 (opens new window)直属“211工程 (opens new window)”高校，是优势学科创新平台项目 (opens new window)高校之一，2012年4月成为首个被国家教育部、国家财政部、北京市共建高校，2017年9月被列入国家双一流计划 (opens new window)一流学科建设高校。

中央财经大学追溯于1949年的中央税务学校，历经中央财政学院、中央财经学院、中央财政金融学院等发展阶段，最终定名为现在的中央财经大学。

# 中国政法大学

中国政法大学，原名北京政法学院，简称法大或政法大学或中政大，中国 (opens new window)法学 (opens new window)类大学，位于北京 (opens new window)，有学院路 (opens new window)和昌平 (opens new window)两个校区，学院路校区位于蓟门桥 (opens new window)畔，昌平校区座落在军都山 (opens new window)下。中国政法大学是中国大陆著名的法学院校，211工程 (opens new window)重点建设高校、一流学科建设高校。被前中国国务院总理 (opens new window)温家宝 (opens new window)以及副总理李岚清 (opens new window)誉为“培养法律人才的最高学府”和“中国法学最高学府”。[1] (opens new window)[2] (opens new window)[3] (opens new window)

# 中国农业大学

中国农业大学是一所以农学 (opens new window)、生命科学 (opens new window)和农业工程 (opens new window)为特色和优势的研究型大学 (opens new window)[1] (opens new window)，是中国现代农业高等教育的起源地。中国农业大学是教育部直属、进入国家“211工程 (opens new window)”和“985工程 (opens new window)”建设的全国重点大学，由教育部、农业部和北京市共同建设[2] (opens new window)。

# 中央民族大学

中央民族大学，简称民大或中央民大，中国 (opens new window)著名高校 (opens new window)，国家“211工程 (opens new window)”、“985工程 (opens new window)”重点建设高校。学校正式创建于1951年，其前身是1941年抗战时期，在延安成立的“民族学院”。1993年，中央民族学院更名为中央民族大学，时任中国共产党中央委员会总书记 (opens new window)江泽民 (opens new window)题写了新校名。位于知名高校林立的北京市海淀区 (opens new window)，南临中国国家图书馆 (opens new window)，北临中关村 (opens new window)。

中央民族大学是中国少数民族 (opens new window)教育的最高学府，是目前中国唯一一所拥有中国五十六个民族 (opens new window)师生员工的高等学校 (opens new window)，并以民族学 (opens new window)、社会学 (opens new window)、文学 (opens new window)、宗教学 (opens new window)的研究著称。截至2011年11月，各类在校生1.5万余名，其中海外留学生374名。

# 中央戏剧学院

中央戏剧学院，简称中戏，是中国戏剧艺术教育的最高学府。1950年4月2日正式成立，校名由毛泽东 (opens new window)题写。前身是延安鲁迅艺术学院 (opens new window)戏剧系、华北大学文艺学院 (opens new window)、南京国立戏剧专科学校 (opens new window)。校址位于北京市 (opens new window)东城区东棉花胡同39号。不少知名艺人都是该校的毕业生。该校和北京电影学院 (opens new window)、上海戏剧学院 (opens new window)、中国戏曲学院 (opens new window)是中国四大知名艺术人材培养学校。

# 中国石油大学

中国石油大学是中华人民共和国 (opens new window)教育部 (opens new window)直属全国重点大学，是国家“211工程 (opens new window)”重点建设院校，是一所以工为主，多学科协调发展的教学研究型大学，是中国最早建立的也是最大的一所石油 (opens new window)高等学府。目前已发展成一所“以工为主、工管理文相结合”的全国重点大学。中国石油大学由中国石油大学（华东） (opens new window)和中国石油大学（北京） (opens new window)两个独立办学机构组成。
