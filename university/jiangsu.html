# 南京大学

南京大学（英语：Nanjing University，缩写为NJU），简称“南大”[注 1] (opens new window)，位于中国 (opens new window)南京市 (opens new window)，是源远流长的高等学府，上可溯至三国 (opens new window)吴 (opens new window)永安元年[4] (opens new window)，历史上曾历经多次变迁[注 2] (opens new window)，亦是中国第一所集教学和研究于一体的现代大学[注 3] (opens new window)。中华民国 (opens new window)政府 (opens new window)撤离南京后，中华人民共和国 (opens new window)成立前夕，由“国立中央大学”易名“国立南京大学”，翌年径称“南京大学”，沿用至今[5] (opens new window)[6] (opens new window)

南京大学在长期的历史中积淀了丰厚的学风传统和精神遗产。南京大学是中国现代科学 (opens new window)的发祥地[注 4] (opens new window)，亦为现代儒家思想 (opens new window)与中华文明 (opens new window)复兴的基地[注 5] (opens new window)，倡行人文思想之会通与学术之昌明以求世界的和平繁荣，在教育、学术和文化上均具重要贡献和影响。[7] (opens new window)

南京大学是研究型综合大学，格物致知，广博易良，向有学科齐备的传统，涵盖了众多领域。现有文学、历史、地理与海洋、地球、大气、天文空间、环境、哲学、数学、物理、化学化工、生、医、政、法、商、社会、信息管理、新闻传播、外国语、工程管理、建筑与规划、电子、计算机、工程与应用科学（含材料、能源、生物医学、量子电子学与光学工程系）等二十多个独立的学科类学院（学系）[注 2] (opens new window)。校园主要有座落在南京市中心的鼓楼校区和位于南京东北部栖霞区的仙林校区，其中仙林校区为本科生主校区，并在逐步成为南大主校区[8] (opens new window)。

南京大学现为中华人民共和国教育部 (opens new window)直属重点大学，属“211工程 (opens new window)”、“985工程 (opens new window)”高校，为基础学科拔尖学生培养试验计划 (opens new window)、九校联盟 (opens new window)成员。南大现有1个国家实验室（筹），7个国家重点实验室，15个部、省重点实验室，21个国家和部、省工程中心[9] (opens new window)，人文社会科学方面有27个各类重点研究基地[10] (opens new window)，此外全校还设有200多个研究机构[11] (opens new window)[12] (opens new window)。医学院有附属鼓楼医院 (opens new window)、金陵医院 (opens new window)（南京总医院 (opens new window)）、南京市口腔医院 (opens new window)。除大学外，还有南大附属中学 (opens new window)、丁家桥小学 (opens new window)等[注 3] (opens new window)。另有科技产业园区和多个产学研结合机构。南京大学的师资队伍包括超过二十位诺贝尔奖得者[13] (opens new window)。 依据评估全球大学的自然科学研究成果的主要指标“自然指数排行榜”(Nature Index 2017 Tables)，南京大学的排名为全国第二（仅次于北京大学）、全亚太地区第三、全球第十二，超越加州理工大学、康奈尔大学、新加坡国立大学、南洋理工大学、伦敦帝王学院等名校[14] (opens new window)。南京大学中文系的排名保持在全国前两名，与北京大学并列第一或仅次于北京大学[15] (opens new window)[16] (opens new window)。除此，南京大学中文系为培养全国第一位中文博士莫砺锋教授的大学[17] (opens new window)。因此，南京大学与北京大学在中文与自然科学领域在学界被公认为全国最卓越的学府。南京大学以门槛高而著称，在“最难进的大学”排名中名列全国第一[18] (opens new window)。在2018 QS世界大学排名中，南京大学位列世界第114位。[19] (opens new window)

# 南京农业大学

南京农业大学（英语：Nanjing Agricultural University）位于中华人民共和国江苏省南京市，是一所以农业和生命科学为优势和特色，兼有农学、理学、经济学、管理学、工学、文学、法学等多学科的教育部直属高等院校，是国家“211工程”重点建设大学和“985优势学科创新平台”高校之一。原党委书记左惟已经调任东南大学党委书记，校长周光宏教授。[1] (opens new window)

# 东南大学

东南大学（英语：Southeast University，缩写：SEU），简称“东大”，位于中华人民共和国 (opens new window)江苏省 (opens new window)南京市 (opens new window)，是中央直管、教育部直属 (opens new window)的研究型综合性大学，原四大工学院 (opens new window)、建筑老八校 (opens new window)之一，入选首批211工程 (opens new window)及首批985工程 (opens new window)重点建设高校[2] (opens new window)，也是中国“双一流 (opens new window)”计划“世界一流大学建设”高校（A类）、“2011计划 (opens new window)”、“111计划 (opens new window)”入选高校，“卓越大学联盟 (opens new window)”、“卓越工程师教育培养计划”、“卓越医生教育培养计划”、“一带一路 (opens new window)”高校联盟[3] (opens new window)、“中英大学工程教育与研究联盟”、“中俄工科大学联盟”、“中欧工程教育平台”、“长三角 (opens new window)高校合作联盟”主要成员。东南大学也是中国国防科工局 (opens new window)、教育部 (opens new window)共建16所部属高校之一，教育部批准的中国首批自主招生 (opens new window)试点高校、建设高水平艺术 (opens new window)团试点高校、建设高水平运动 (opens new window)队试点高校，拥有中国科技部 (opens new window)、教育部首批试点及首批授牌的国家大学科技 (opens new window)园。

东南大学是20世纪初中国最早建立的现代高等学府之一，2012年中国国务院总理温家宝 (opens new window)为东南大学建校110周年题词“以科学名世，以人才报国”[4] (opens new window)。早在1920年代，东南大学首任校长郭秉文 (opens new window)主张“自由讲学”，延揽了国内外许多名流学者到此讲学，如美国杜威 (opens new window)博士讲授《实验教育哲学》；美国吴卫士博士讲授《昆虫学 (opens new window)》；美国孟禄博士讲授《教育学 (opens new window)》；德国杜里舒博士讲授《生机哲学 (opens new window)》；北大胡适 (opens new window)博士讲授《实用主义 (opens new window)》；梁启超 (opens new window)教授讲授《先秦政治思想史》；江亢虎 (opens new window)博士讲授《劳动问题》；张东荪 (opens new window)教授讲授《新闻学大意》；还有本校常务校董和工科教授杨杏佛 (opens new window)的《政治改造思想》等，此外还延聘了佛学大师欧阳竟无 (opens new window)讲了一次《佛法 (opens new window)非宗教非哲学而为世人所必需》的讲座[5] (opens new window)。1923年国学大师梁启超 (opens new window)在东南大学讲学时就称“或许我们这里，东南大学，就是推行美化的大本营。”（美化即美国式的教育）[6] (opens new window)。

东南大学依据“打造强势工科 (opens new window)、强化优势理科 (opens new window)、发展精品文科 (opens new window)、建设特色医科 (opens new window)”的一流学科发展战略多学科协调发展，在2017年9月教育部公布的“世界一流大学和一流学科建设 (opens new window)”计划中，东南大学入选11个世界一流学科建设点（数量居中国内地高校第8位）；在2017年12月教育部公布的第四次全国学科评估中，东南大学5个学科获评“A+”顶级学科（数量亦居中国内地高校第8位）。现有10个国家重点实验室 (opens new window)、国家工程实验室、国家专业实验室和国家工程（技术）研究中心，16个教育部重点实验室和教育部工程研究中心[2] (opens new window)，30个其他省部级重点实验室、工程（技术）研究中心、人文学科 (opens new window)和社会科学 (opens new window)研究中心、重点智库 (opens new window)，以及东南大学建筑 (opens new window)研究所、太赫兹 (opens new window)研究所、医学影像 (opens new window)与精确诊治技术研究中心、学习 (opens new window)科学研究中心、空间科学 (opens new window)与技术研究院（阿尔法磁谱仪 (opens new window) AMS研究中心）、汽车工程 (opens new window)研究院、生命科学 (opens new window)研究院、城市 (opens new window)工程科学技术研究院、南京通信技术 (opens new window)研究院、智能 (opens new window)运输系统（ITS）研究中心、现代计算机集成制造 (opens new window)系统(CIMS)研究中心、集团经济 (opens new window)与产业组织研究中心等近百个学术科研机构。

东南大学目前在美国 (opens new window)和白俄罗斯 (opens new window)设有3个孔子学院 (opens new window)，同时与全球100多所大学和研究机构签订了合作交流协议，在校留学生 (opens new window)人数达到1947人，其中学历留学生1492人，留学研究生536人，学历留学生和留学研究生 (opens new window)数均居中国江苏省 (opens new window)第一。东南大学—蒙纳士大学 (opens new window)苏州 (opens new window)联合研究生院是教育部批准的第一个中外联合研究生院 (opens new window)[2] (opens new window)。曾任东南大学校长的韦钰 (opens new window)院士是1978年改革开放 (opens new window)后第一批留学生代表，作为中国第一位电子学女博士，德国 (opens new window)学成回国后，在东南大学创建了中国第一个分子与生物电子学实验室（现名吴健雄 (opens new window)实验室）及中国第一个“生物电子学”（生物医学工程 (opens new window)）博士点[7] (opens new window)。

# 江南大学

江南大学坐落于江苏省无锡市，是中华人民共和国教育部直属的一所具有轻工特色的高校，是国家“211工程”和“985工程优势学科创新平台”重点建设大学，国家首批“双一流”世界一流学科建设高校[1] (opens new window)，高水平行业特色大学优质资源共享联盟成员，入选“111计划”、“千人计划”，是首批“卓越农林人才教育培养计划”、首批“卓越工程师教育培养计划”改革试点高校，被誉为“轻工高等教育明珠”。[2] (opens new window)
