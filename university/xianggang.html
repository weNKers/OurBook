# 香港大学

香港大学，简称港大（英语：The University of Hong Kong，缩写 (opens new window)：HKU），是香港 (opens new window)一所公立研究型大学 (opens new window)，大学本部位于香港岛 (opens new window)的薄扶林 (opens new window)[3] (opens new window)[注 1] (opens new window)。香港大学成立于1911年，并于1912年3月11日正式办学，[6] (opens new window)[7] (opens new window)是香港最早建立的高等教育 (opens new window)机构。成立之初，更是大英帝国 (opens new window)在东亚 (opens new window)成立的唯一一间大学[8] (opens new window)。

香港大学创立于殖民地时期 (opens new window)的香港，其溯源是1887年成立的华人西医书院 (opens new window)，该校后来并入香港大学并成为香港大学医学院 (opens new window)，与文学院 (opens new window)及工程学院一同被认为是香港大学的创校学院。香港大学在日占时期 (opens new window)遭受严重破坏，太平洋战争 (opens new window)结束后，香港大学于1948年恢复运作，其后多所学院及学术部门相继成立[7] (opens new window)。港大创校以来一直采用英文 (opens new window)教学，香港大学现在共有十所学术学院，香港大学作为跨学术领域的综合大学，其以法律学 (opens new window)、政治学 (opens new window)、教育学 (opens new window)、工程学 (opens new window)、会计学 (opens new window)、生命科学 (opens new window)及医学 (opens new window)见长[9] (opens new window)[10] (opens new window)[11] (opens new window)[12] (opens new window)[13] (opens new window)。港大医学院 (opens new window)是全球第一个成功鉴定及上报冠状病毒 (opens new window)（非典型肺炎 (opens new window)病原体 (opens new window)）的研究单位[14] (opens new window)。香港大学自创校以来一直为香港社会培养出各界的著名人士，香港大学的排名在香港高等院校中长期名列前茅，亦曾持续位于《QS (opens new window)》及《泰晤士 (opens new window)》综合排名的亚洲三甲之列[15] (opens new window)[16] (opens new window)[17] (opens new window)。港大牙医学院 (opens new window)于2016年及2017年全球排名第一[18] (opens new window)，而港大教育学院 (opens new window)于2017年为亚洲第一[19] (opens new window)。

# 香港中文大学

香港中文大学（英语：The Chinese University of Hong Kong，缩写 (opens new window)为 CUHK），简称中文大学、中大[注 1] (opens new window)，是一所坐落于香港 (opens new window)沙田 (opens new window)马料水 (opens new window)的公立 (opens new window)研究型 (opens new window)大学。

香港中文大学是清末 (opens new window)至民国时期 (opens new window)的顶级大学在香港的延续。创校教授为民国时期著名大儒、专家和国民政府 (opens new window)及行宪后中华民国政府 (opens new window)要员。大学所合并的三所书院[4] (opens new window)中的崇基学院 (opens new window)和联合书院 (opens new window)前身为1845年起始的清末 (opens new window)至中华民国 (opens new window)时期在中国大陆地区 (opens new window)所建立的享有极高声誉的教会 (opens new window)及私立大学 (opens new window)，是香港历史源流上最远久的高等学院。

香港中文大学于1963年合并香港高等学校而成立，并于1966年开办香港首间研究院 (opens new window)，是香港第一所研究型大学 (opens new window)[5] (opens new window)，也是香港唯一采用书院联邦制 (opens new window)办学的大学，现共有9所成员书院及8所学术学院。中大的创立打破了英国殖民地 (opens new window)只允许一所高等学府存在的惯例，并成功掀起香港1970年代 (opens new window)的“中文运动 (opens new window)”，具历史及时代意义。[6] (opens new window)[7] (opens new window)[8] (opens new window)[9] (opens new window)虽然最初的大学条例将中文列作指定教学语言，[10] (opens new window)但校方目前提供的大部分科目均以英语 (opens new window)教授，只有某些部门同时提供粤语 (opens new window)及标准汉语 (opens new window)指引。[11] (opens new window)

该校既是香港的天灾 (opens new window)监测中心，亦是该地社会科学 (opens new window)、国学 (opens new window)、工商管理 (opens new window)、理学 (opens new window)及医学 (opens new window)的主要研究机构。[12] (opens new window)其曾经有3名教授 (opens new window)和1名校长 (opens new window)获得诺贝尔奖 (opens new window)，校友 (opens new window)衔盖菲尔兹奖 (opens new window)、沃尔夫数学奖 (opens new window)、勒诺多文学奖 (opens new window)和克鲁格人文与社会科学终身成就奖得主 (opens new window)以及包括岭南大学 (opens new window)、香港大学 (opens new window)等在内的等高校的校长，系唯一具有获得国际顶级奖项的校友以及诺贝尔奖、菲尔兹奖及图灵奖 (opens new window)得主专任任教的香港高校。[13] (opens new window)香港教育历来产生的诺贝尔奖等级的学者中，有67%是中大的校友、教授（参见主栏目香港教育 (opens new window)）。

香港中文大学本科毕业生获得罗德奖学金 (opens new window)（誉为本科“诺贝尔奖”）的数目为香港最高[14] (opens new window)，也是历代神童出现最多的本地大学[15] (opens new window)，其顶尖科系普遍收生成绩为香港八所大学中最高。[16] (opens new window)

# 香港城市大学

香港城市大学（简称城大；英语：City University of Hong Kong），是一所公立应用型大学，位于香港 (opens new window)九龙 (opens new window)深水埗区 (opens new window)九龙塘 (opens new window)。前身为于1984年创立的“香港城市理工学院”，在开校时的校舍设于旺角的工业贸易署大楼 (opens new window)之内，到1990年迁入九龙塘朱牯仔的永久校舍，于1994年升格为香港城市大学。

香港城市大学是八间由大学教育资助委员会 (opens new window)（教资会）资助的大学之一，注重多元化、跨文化学习与全球视野。香港城市大学学术水平近年不断显著提升，更连续多年获国际评为香港四所最佳高等学府之一。城大由商学院、人文社会科学院、科学及工程学院、动物医学及生命科学院、创意媒体学院、能源及环境学院及法律 (opens new window)学院7个学院、25个学系、37个科研中心组成[5] (opens new window) ，提供逾100项学士 (opens new window)学位及研究生 (opens new window)课程。 周亦卿研究生院负责所有研究生 (opens new window)课程的运作及发展，及提升研究生 (opens new window)的学习环境。[6] (opens new window)[7] (opens new window)

城大在QS综合排名位处前列[8] (opens new window)[9] (opens new window)。于2017年，香港城市大学在QS世界大学排名 (opens new window)位列全球第49位[10] (opens new window)，香港第4。至2018年，香港城市大学的排名进一步上升，增至亚洲第8名[11] (opens new window)，香港第3[12] (opens new window)，并于同年世界大学排名榜入围世界50强、在全球50强年轻大学位列第4位[13] (opens new window)，香港八间教资会资助院校中有七间同时打入QS世界大学排名及QS亚洲大学排名，城大是香港七间登上两个QS综合排名的大学之一，更在两个QS综合大学排名均在前列[14] (opens new window)。

在2017年公布的泰晤士高等教育世界大学排名 (opens new window)，在全球校龄50年以下的年轻大学当中，香港城市大学名列全球第7位[15] (opens new window)。

# 香港理工大学

香港理工大学（英语：The Hong Kong Polytechnic University；缩写：PolyU或HKPU），简称理大，是一所坐落于香港 (opens new window)九龙 (opens new window)红磡湾 (opens new window)的公立 (opens new window)应用研究型大学。该校前身可追溯至1937年，并于1994年正名为香港理工大学，是八间大学教育资助委员会 (opens new window)（教资会）资助的法定公立大学之一。

根据泰晤士高等教育世界大学排名 (opens new window)，理大于2017-2018位列全球第182位[6] (opens new window)、亚洲第19位[7] (opens new window)及全球200强年轻大学第20位[8] (opens new window)。至于QS世界大学排名 (opens new window)，理大位列全球第95位[9] (opens new window)、亚洲第27位[10] (opens new window)[11] (opens new window)，香港八间教资会资助院校中有七间同时打入QS世界大学排名及QS亚洲大学排名[12] (opens new window)，理大是香港七间登上两个QS综合排名的大学之一，而且位处排名前列[13] (opens new window)。理大是全球50强年轻大学第6位[14] (opens new window)[15] (opens new window)。在地区认受性方面，按2004-2017年香港大学民意研究计划 (opens new window)调查结果，理大在“港人眼中的大专院校排名”中名列第4 [16] (opens new window)，亦于2016年“本港雇主最愿意聘用的大学毕业生”调查中名列第2[17] (opens new window)。

截至2014年8月，香港理工大学拥有工程 (opens new window)、建设 (opens new window)及环境 (opens new window)、工商管理 (opens new window)、医疗 (opens new window)及社会科学 (opens new window)、人文 (opens new window)、应用科学及纺织、设计 (opens new window)及酒店及旅游业管理8个学院 (opens new window)，27个学系 (opens new window)，20余个科研中心。大学的课程及研究均以应用为本，并与工商业界保持紧密联系，而计算理工大学提供的深造课程、学士学位及非学位课程让其学生人数为全港所有教资会资助大学中最多的，历年来的毕业生已多逾33万。

大学在创新科研簵畴上主要配合社会需要，亦有与国际其他院校联结和合作，如环保 (opens new window)、医疗产品等都曾获得奖项[18] (opens new window)[19] (opens new window)。大学在太空技术的发展也有成就[20] (opens new window)，早于1995年已为俄罗斯 (opens new window)的太空计划提供设备，在嫦娥三号 (opens new window)探月航天器也装有由理工大学研发的仪器[21] (opens new window)。

# 香港浸会大学

香港浸会大学（英语：Hong Kong Baptist University，缩写HKBU），是一间有基督教 (opens new window)浸信会 (opens new window)背景的香港法定公立 (opens new window)大学 (opens new window)，浸大作为研究型博雅大学[2] (opens new window)[3] (opens new window)，重视教学、研究和社会服务[4] (opens new window)，校内设有与中国科学院 (opens new window)合作建设的环境与生物分析国家重点实验室[5] (opens new window)，推动科学研究发展。根据泰晤士高等教育世界大学排名 (opens new window)，浸大于2018年位列亚洲第55名[6] (opens new window)，成为亚洲最佳前100所高等学府之一。至于在2018年QS世界大学排名 (opens new window)，浸大位列全球第299名[7] (opens new window)[8] (opens new window)[9] (opens new window)。在2018年QS世界大学的分科排名中，浸大地理系位列全球151-200名，宗哲系亦位列全球151-200名[7] (opens new window)。而在2016年世界大学学术排名 (opens new window)数学系位列全球151-200名，而计算机科学系更位列全球101-150名[10] (opens new window)。传理学院最负盛名，辖下新闻系 (opens new window)在2011年经Asian Correspondent (opens new window)新闻网评定为“亚洲学生心中的全球十大新闻学府”[11] (opens new window)，和榜内的哥伦比亚大学 (opens new window)新闻学院齐名，乃校内招生志愿的前三名[12] (opens new window)。于九龙仔 (opens new window)、毕架山 (opens new window)、平山 (opens new window)与石门 (opens new window)均有校园，简称浸大。

# 香港教育大学

香港教育大学（简称：教大；英语：The Education University of Hong Kong，缩写 (opens new window) EdUHK）是香港唯一以师范 (opens new window)教育为本的大学，亦是八间大学教育资助委员会 (opens new window)辖下的法定公立大学之一。

教大前身香港教育学院为1994年合并五间师训学院而成，他们分别为[3] (opens new window)[4] (opens new window)：

  1. 罗富国师范学院 (opens new window) （英语：Northcote College of Education，于 1939年创立）
  2. 葛量洪教育学院 (opens new window) （英语：Grantham College of Education，于 1951年创立）
  3. 柏立基教育学院 (opens new window) （英语：Sir Robert Black College of Education，于 1960年创立）
  4. 香港工商师范学院 (opens new window) （英语：Hong Kong Technical Teachers' College，于 1974年创立）
  5. 语文教育学院 (opens new window) （英语：Institute of Language in Education，于 1982年创立）

2015年9月，大学教育资助委员会 (opens new window)经评审认为教院采取的内部学术流程，其质素与其他教资会资助大学并无二致，而在院校层面，如管理、研究、毕业生水平等方面亦与其他教资会大学比肩，反映其已合乎资助大学水平[5] (opens new window)。因此。通过向行政会议建议让香港教育学院正名大学。2016年1月，行政长官会同行政会议 (opens new window)批准向教院授予大学名衔。2016年5月，立法会三读通过《2016年香港教育学院（修订）条例草案》[6] (opens new window)，于同月刊宪生效，“香港教育学院”正式改名为“香港教育大学”，成为香港第十间大学及第八间公帑资助大学。

2018年，教大的教育及师资培训在国际QS世界大学排名 (opens new window)中位列全球第9名[7] (opens new window)、亚洲区第2名[8] (opens new window)；语言学则位列全球首151-200名 [9] (opens new window)。教大发展时间较短的心理学和社会科学及管理也分别于2018年在国际QS世界大学排名 (opens new window)中取得全球第251-300名[10] (opens new window)和第323名[11] (opens new window)。

在地区认受性方面，按2004-2016年香港大学民意研究计划 (opens new window)调查结果，教大在“港人眼中的大专院校排名”中名列第7 [12] (opens new window)[13] (opens new window)，亦于2016年“本港雇主最愿意聘用的大学毕业生”调查中名列第7[14] (opens new window)。
